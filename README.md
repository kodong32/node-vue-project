# 🧩 발달장애 케어 시스템

## 📌 프로젝트 소개
발달장애인, 보호자, 기관 담당자, 시스템 관리자가 사용할 수 있는 통합 지원 관리 시스템입니다.  
사용자·기관 정보, 조사지, 지원계획서, 지원 이력 등을 통합 관리하며, 역할별 권한에 따라 기능을 제공합니다.

---

## 🎯 개발 목적
- 발달장애인과 보호자, 기관 간의 지원 정보를 효율적으로 관리
- 역할별 접근 제어를 통해 사용자 유형에 맞는 기능 제공
- 조사지 및 지원계획서를 통한 체계적인 지원 관리 시스템 구축

---

## 💡 기대 효과
- 데이터 기반의 체계적인 지원 관리 가능
- 기관과 보호자 간 정보 전달 효율 향상
- 발달장애인의 삶의 질 향상 및 사회성 증진
